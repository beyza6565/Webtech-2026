Die Dailyhabits App ist eine Webanwendung, die Nutzern täglich zufällige Aufgaben („Challenges“) bereitstellt. Ziel ist es, den Alltag abwechslungsreicher zu gestalten und Nutzer zu motivieren, kleine Aktivitäten aus verschiedenen Bereichen wie Fitness, Lernen oder soziale Interaktion umzusetzen bzw.langfristig positive Gewohnheiten zu entwickeln.

Funktionen
Generierung einer zufälligen Challenge
Markieren von Challenges als erledigt
Anzeige des Fortschritts (z. B. abgeschlossene Aufgaben)
Auswahl von Kategorien (z. B. Fitness, Lernen)
Möglichkeit, eigene Challenges hinzuzufügen
Streak-System zur Motivation (aufeinanderfolgende Tage)
